# calculo-de-volatilidade-opcoes
esse código gera uma analise das volatilidades de uma determinada ação da B3 inicialmente, apos isso, inserindo os strikes, preços, vencimentos e categoria das opcoes ele analisa a volatilidade implicita e compara com a volatilidade historica, usando isso para indicar se a opcao esta cara ou barata, alem disso diz quais os skews das puts e calls
